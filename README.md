# 심리검사 시스템 🧠

완전한 기능을 갖춘 웹 기반 심리검사 플랫폼으로, 상담사와 내담자 간의 원격 심리 평가를 지원합니다.

## 🌐 배포된 URL

**개발 환경:** https://3000-in4fm39zbf1k3ggufr0fp-a402f90a.sandbox.novita.ai

## ✨ 주요 기능

### 🕊️🧠 상담 유형 선택 및 맞춤형 AI 분석 (NEW! v1.6.0)
- **🧠 심리상담**: 심리학 이론과 과학적 접근 기반 분석
  - **이론적 배경**: 인지행동치료(CBT), 정신분석, 인본주의 심리학, Bowen 가족치료
  - **분석 특징**: 자기 효능감, 애착 이론, 감정 조절, 대처 자원 강화
  - **권장사항**: 
    - 정기적 심리 상담 (주 1회)
    - 이완 훈련 및 마음챙김 명상
    - 인지 재구성 및 행동 활성화
    - 감정 일지 작성 및 자기 탐색
  
- **🕊️ 성경적 상담**: 성경 말씀과 기독교 신앙 기반 분석
  - **이론적 배경**: 성경 원리, 복음 중심 상담, 영적 성장
  - **분석 특징**: 
    - 성경 구절 인용 및 적용 (50개 이상의 관련 구절)
    - 그리스도 안에서의 정체성 확립
    - 용서, 은혜, 회개의 관점
    - 하나님의 사랑과 돌보심 중심
  - **권장사항**: 
    - 매일 성경 읽기와 기도 (QT)
    - 성경적 상담 및 영적 멘토링
    - 소그룹/셀 모임 참여
    - 찬양과 경배를 통한 영적 평안
    - 금식기도 및 중보기도

**작동 방식**:
1. 상담사가 링크 생성 시 상담 유형 선택 (성경적/심리)
2. 내담자가 검사 완료
3. AI 분석 생성 시 자동으로 선택된 상담 유형에 맞는 분석 제공
4. 결과 화면에 상담 유형 배지 표시 (🕊️ 성경적 상담 / 🧠 심리상담)

**성경적 상담 분석 예시**
```
[어머니 관계 분석]
어머니와의 관계에서 하나님의 사랑과 돌보심이 반영되어 있습니다. 
'어머니가 자식을 위로함같이 내가 너희를 위로하리니'(이사야 66:13)라는 
말씀처럼, 건강한 어머니상은 하나님의 사랑을 경험하는 통로가 됩니다.

[권장사항]
• 매일 성경 읽기와 기도로 하나님과의 관계 깊이하기
• 성경적 상담을 통해 관계 회복과 용서의 과정 경험하기
• 소그룹이나 셀 모임에서 영적 지지 받기
• 정기적인 교회 출석과 말씀 사역 참여
```

**심리상담 분석 예시**
```
[어머니 관계 분석]
어머니와의 관계가 긍정적으로 형성되어 있습니다. 
애착 관계가 안정적이며, 이는 대인관계 형성의 긍정적 기반이 됩니다.

[권장사항]
• 정기적인 심리 상담을 통한 감정 표현 및 해소
• 인지행동치료(CBT) 기법을 통한 사고 패턴 개선
• 이완 훈련 및 마음챙김 명상 실천
```

### 🕐 데이터 보안 및 관리 (v1.4.0)
- **⏱️ 24시간 자동 삭제**: 검사 결과는 24시간 후 자동 삭제
- **💾 로컬 저장**: JSON 파일로 검사 결과 다운로드 (영구 보관 가능)
- **📂 불러오기**: 저장된 JSON 파일을 언제든지 불러와서 상담 진행
- **⏱️ 실시간 카운트다운**: 남은 시간 실시간 표시 (시:분:초)
- **⚠️ 만료 경고**: 삭제 임박 시 시각적 경고 (6시간 미만 주황색, 3시간 미만 빨간색)

**사용 시나리오**
```
1. 내담자가 검사 완료 → 서버에 24시간 저장
2. 상담사가 "💾 저장" 버튼으로 JSON 다운로드 → 로컬에 영구 보관
3. 24시간 경과 → 서버에서 자동 삭제 (개인정보 보호)
4. 상담 시 "📂 불러오기"로 저장한 JSON 불러오기 → 검사 결과 확인하며 상담 진행
```

### 1️⃣ 검사 유형
- **📝 SCT (문장완성검사)**: 50개 문항으로 구성된 심리 평가
  - 14개 카테고리별 분석 (어머니/아버지 태도, 가족, 이성, 친구, 권위자, 두려움, 죄책감, 능력, 과거/미래, 목표)
  - **✅ 카테고리별 AI 심리 분석 기능**
- **🔍 DSI (자아분화검사)**: 36개 문항의 5점 척도 검사
  - 5개 영역 분석 (인지적 기능, 자아통합, 가족투사, 정서적 단절, 가족퇴행)
  - 총점 기반 수준 평가 (180점 만점)
  - **✅ AI 기반 종합 상담 권장사항 제공**

### 5️⃣ AI 심리 분석 기능 ✅ (NEW!)

**룰 기반 AI 시스템**
- 외부 API 불필요 (완전 독립 실행)

- 심리학적 이론 기반 해석

**SCT 분석 기능**
- 각 카테고리별 개별 분석 생성
- 키워드 패턴 인식 및 감정 분석
- 긍정/부정 요소 자동 탐지

**DSI 분석 기능**

- 5개 영역별 상세 해석
- 강점/약점 영역 식별
- 취약 영역 개선 방안 제시
- Bowen 가족치료 이론 기반 권장사항
- 단기/장기 목표 설정
- 상담 접근법 추천

**분석 예시**
```
[인지적 기능 분석]
감정 조절과 의사결정 능력이 우수합니다. 
충동성이 낮고 논리적 사고가 가능합니다.

[권장사항]
• 정기적인 심리 상담을 통한 감정 표현 및 해소
• 이완 훈련 및 마음챙김 명상 실천
• 자기 탐색 활동 및 가치관 명료화 작업

[단기 목표 (1-3개월)]
• 주 1회 정기 상담 참여
• 감정 일지 작성 (일일)
• 이완 훈련 실천 (주 3회)
```

   - JSON 파일로 다운로드: `검사결과_SCT_세션ID_타임스탬프.json`
   - 상담사/관리자 대시보드에서 "📂 불러오기" 클릭
   - 저장된 JSON 파일 선택 → 검사 결과 화면 자동 표시
   - 삭제된 검사도 JSON 파일이 있으면 언제든지 열람 가능

   - 앱 시작 시 만료된 검사 자동 삭제
   - 1분마다 백그라운드에서 만료 체크

   - 만료된 항목은 회색 처리 및 "삭제됨" 표시


## ⚠️ 주의사항

### LocalStorage 사용 시 유의사항
1. **브라우저별 독립 저장**
   - Chrome과 Safari는 별도의 저장소 사용
   - 같은 브라우저에서만 데이터 공유 가능

2. **캐시 삭제 주의**
   - 브라우저 캐시/쿠키 삭제 시 LocalStorage도 함께 삭제됨
   - 주기적으로 백업 권장

3. **용량 제한**
   - 브라우저당 약 5-10MB 제한
   - 대량 데이터 저장 시 주의 필요

4. **보안**
   - 민감한 정보는 암호화하여 저장 권장
   - 개발자 도구에서 내용 확인 가능
